TwitterFacebookShare
============================================================================

# Features:
Send messages out to the social networks which are deeply integrated into iOS: Twitter, Facebook, using single sign-on on behalf of a user.

# Development technologies:
- Social.framework
- Autolayout

# Reason for creation:
- Self aducation
